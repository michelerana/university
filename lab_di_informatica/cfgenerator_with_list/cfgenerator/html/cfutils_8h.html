Genera il codice fiscale delle persone presenti nella lista.

Data una lista di nodi del tipo node contenente i dati anagrafici delle persone, la scorre e genera il codice fiscale di ognuno di questi a partire dai dati anagrafici letti. char* read_birth_place_code ( FILE *  places_file,
const char *  birth_place 
)

Legge il codice catastale di un comune.

Dato un puntatore allo stream in lettura places_file associato al file dei codici catastali italiani (fornito dal docente) e una stringa contenente il nome del comune birth_place, scorre lo stream fino a quando il nome del comune letto non coincide con il nome del comune birth_place. Se la stringa indicante il comune non viene trovata il puntatore al codice catastale viene impostato a NULL (quindi l'esecuzione si interrompe a causa della post-condizione).

Questa funzione viene utilizzata in generate_cfs al fine di fornire direttamente il codice catastale alla funzione che genera il codice fiscale Per l'implementazione si consiglia di utilizzare le funzioni della libreria standard string.h come strtok(), strcmq() e la strcpy(); queste fanno uso e ritornano puntatori, pertanto utilizzare adeguatamente la malloc(). Ricordarsi inoltre che lo stream viene letto sequenzialmente; la lettura riprende da dove è stata interrotta all'ultimo accesso.

Parametri
[in]places_filepuntatore a stream in lettura collegato al file dei comuni con i codici catastali
[in]lastringa con il nome del comune di nascita
Restituisce
la stringa del codice catastale del comune di nascita
Precondizione
puntatore a stream places_file non NULL
birth_place stringa non vuota
Postcondizione
il puntatore alla stringa del codice catastale ritornato non NULL
il codice catastale restituito deve essere una stringa di 4 caratteri
